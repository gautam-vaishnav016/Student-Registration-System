## Student Registration System

This project demonstrates CRUD operations, client-side validation, and responsive UI development using Tailwind CSS.
A modern and responsive **Student Registration System** built using **HTML, Tailwind CSS, and JavaScript**.  
This project allows users to **add, edit, delete, and manage student records** with data persistence using **localStorage**.

---

## Features

- Add new student records
- Edit existing student details
- Delete student records
- Data persistence using **localStorage**
- Input validation using **JavaScript & Regex**
- Fully responsive (Mobile, Tablet, Desktop)
- Modern UI using **Tailwind CSS**
- Scrollable student records table
- Prevents empty or invalid submissions

---

## Technologies Used

- **HTML5** – Structure
- **Tailwind CSS (CLI)** – Styling & Responsiveness
- **JavaScript** – Functionality & Logic
- **Local Storage** – Data persistence
